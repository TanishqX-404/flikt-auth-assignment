System of Registration and Login

This project, which is created as part of internship assignment, is a frontend authentication interface.


#Summary
The application offers a simple, dark-themed user interface along with client-side validation for login and registration forms.

## Features: Registration and login forms
Client-side form validation; password length validation; form switching without page reload; a responsive and tidy dark user interface; and JavaScript DOM manipulation

## Tech Stack: JavaScript,HTML5,CSS3
